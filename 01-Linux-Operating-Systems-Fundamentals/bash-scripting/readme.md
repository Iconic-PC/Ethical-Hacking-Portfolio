# Bash Scripting Exercises

This folder contains basic Bash scripting exercises completed as part of Linux Operating Systems Fundamentals training in cybersecurity.

## Scripts Included

- user_input.sh → Collects and processes user input
- password_checker.sh → Basic password validation logic
- countdown.sh → Simple countdown automation script
- sum_function.sh → Demonstrates arithmetic operations in Bash

## Skills Demonstrated

- Bash scripting fundamentals
- Variables and user input handling
- Conditional statements
- Loops and basic logic
- Command-line automation

## Security Relevance

Bash scripting is essential in cybersecurity for:
- automating reconnaissance tasks
- log parsing
- system monitoring
- penetration testing workflows

## Note

All scripts were executed in a controlled Kali Linux environment for educational purposes only.
